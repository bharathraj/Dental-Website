# Radiant Dental — Premium Dental Clinic Website

A world-class, award-winning dental clinic website built with **Next.js 15**, **React 19**, **Tailwind CSS**, and **Framer Motion**.

## ✨ Features

- 🎨 **Premium Design** — Royal Blue + Teal + White palette with glassmorphism
- 🏃 **Framer Motion Animations** — Scroll reveals, hover effects, page transitions
- 📱 **Fully Responsive** — Mobile-first design with hamburger menu
- 🦷 **12 Treatment Cards** — With hover animations and gradient accents
- 👨‍⚕️ **Doctor Profiles** — Real photography with booking CTAs
- ⭐ **Patient Testimonials** — Google-style review cards
- 📅 **Appointment Form** — Glassmorphism booking form with validation
- ❓ **FAQ Accordion** — Smooth expand/collapse with AnimatePresence
- 📸 **Smile Gallery** — Interactive before/after slider
- 💬 **Floating WhatsApp** — Always-visible chat button
- 🔝 **Back to Top** — Appears on scroll
- 🏥 **Insurance Partners** — Placeholder logos
- 📝 **Blog Section** — Health insights with hover effects
- 📊 **Animated Counters** — Stats that count up on scroll

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## 📁 Project Structure

```
radiant-dental/
├── app/
│   ├── components/     # Reusable components (Navbar, FloatingButtons, etc.)
│   ├── sections/       # Page sections (Hero, Treatments, Doctors, etc.)
│   ├── hooks/          # Custom React hooks
│   ├── lib/            # Data and utilities
│   ├── globals.css     # Global styles & Tailwind
│   ├── layout.tsx      # Root layout with metadata
│   └── page.tsx        # Home page
├── public/             # Static assets
├── tailwind.config.ts  # Tailwind configuration
└── next.config.js      # Next.js configuration
```
